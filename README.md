# Proyecto: Analisis de embudo y retencion para MercadoLibre Bootcamp Data Analyst TripleTen 2026

 ## Objetivo
 
Proyecto orientado al análisis del comportamiento de usuarios en MercadoLibre mediante la construcción de embudos de conversión y análisis de retención. Se utilizaron consultas SQL avanzadas para identificar puntos de abandono, medir la conversión entre etapas y evaluar la permanencia de los usuarios a lo largo del tiempo.

- Proyecto _Análisis de embudo y retención para MercadoLibre - Retencion x Pais.csv
  
- Proyecto _Análisis de embudo y retención para MercadoLibre - Retencion x Cohort.csv
  
- Proyecto _Análisis de embudo y retención para MercadoLibre - Resumen ejecutivo.pdf

## Actividades realizadas

•	Construcción de funnels con CTEs

•	Análisis de cohortes y retención

•	Creación de heatmaps

•	Desarrollo de embudos de conversión

## Herramientas

•	SQL

•	CTEs

•	Cohortes

•	Funnels

## Resultados

•	Identificación de las etapas con mayor pérdida de usuarios

•	Medición de tasas de conversión entre eventos del funnel

•	Creación de cohortes para evaluar retención por periodo

•	Elaboración de heatmaps para visualizar patrones de permanencia
